# Skyhook

Skyhook is a (command line, for now) utility for backing up files in Amazon's S3 cloud storage service. 

Directories that are backed up are defined along with other settings in a YAML configuration file.

## Installation

Requirements:
- Ruby 1.9.x
- Rubygems
- Fog gem
	gem install fog
